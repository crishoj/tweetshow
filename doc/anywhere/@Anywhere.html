@Anywhere JavaScript API

User

Class Methods
find(identifer) → User id or screen_name
findAll(identifiers) → [Users] ids and/or screen_names
current → User the currently logged in user
search(query) → [Users] search for a user
Instance Properties
id, name, screenName, location, description, profileImageUrl, url, protected, followersCount, profileBackgroundColor, profileTextColor, profileLinkColor, profileSidebarFillColor, profileSidebarBorderColor, friendsCount, createdAt, favoritesCount, utcOffset, timeZone, profileBackgroundImageUrl, profileBackgroundTile, statusesCount, notifications, following, verified
Special Instance Properties
status→ Status the user's latest status
Instance Methods
favorites → [Statuses] the user's favorited statuses
userTimeline → [Statuses] the user's user timeline (profile page on twitter.com)
followers → [Users] users that follow this user
friends → [Users] the users that this user follows
isFollowing → true | false is the logged in user following this user?
isFriend → true | false alias to User#isFollowing
isFollowedBy → true | false does this user follow the logged in user?
isMutual → true | false does the logged in user follow this user and vice-versa?
block → User, block this user for the logged in user
unblock → User, unblock this user for the logged in user
directMessage(message) → DirectMessage send a direct message to this user from the logged in user
dm(message) → DirectMessage alias to User#directMessage
follow → User as the logged in user, follow this user
unfollow → User as the logged in user, unfollow this user
report → User as the logged in user, report this user for spam
notifications
(true|false|"on"|"off") → User		 as the logged in user, receive device updates when this user tweets(requires phone setup)
lists → [Lists] lists this user has created
memberships → [Lists] lists that follow this user
subscriptions → [Lists] lists that this user follows
isVerified → true | false is this user verified? (getter for 'verified' instance property)

Facade method examples

    <script src="jquery.js"></script>
    <script>
    twttr.anywhere(function(T) {

      // get @ded's first 20 statuses, filter only the tweets that mention photography, and render each into an HTML element
      T.User.find('ded').timeline().first(20).filter(filterer).each(function(status) {
        $('div#tweets').append('<p>' + status.screenName + ': ' + status.text + '</p>');
      });

      function filterer(status) {
        return status.text.match(/photography/);
      }

    });
    </script>
